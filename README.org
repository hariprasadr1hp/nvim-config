#+title: nvim-config

** Overview
nvim customized for c++ and python development

** Install ~neovim~
*** macos
#+begin_src bash
brew install neovim
#+end_src

*** ubuntu
#+begin_src bash
sudo add-apt-repository ppa:neovim-ppa/unstable
sudo apt update
sudo apt install neovim
#+end_src

** Setting up
+ clone the repo as =~/.config/nvim=
#+begin_src bash
cd ~/.config/nvim
touch vimscript/temp.vim
touch lua/temp.lua
#+end_src
+ inside neovim
  - ~:Lazy~
